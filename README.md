# Hello Test PWA

Tiny installable app for iPhone and Windows.

## Use

Open the GitHub Pages URL in Safari or a desktop browser.

On iPhone:

1. Open the public GitHub Pages URL in Safari.
2. Tap Share.
3. Tap `Add to Home Screen`.
4. Launch `Hello Test` from the new home screen icon.

On Windows:

1. Open the public GitHub Pages URL in Microsoft Edge or Chrome.
2. Use the browser install button, or open the browser menu and choose `Apps` -> `Install this site as an app`.
3. Launch `Hello Test` from the Start menu.
